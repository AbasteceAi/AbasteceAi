# AbasteceAi
O AbasteceAí tem como intuito elucidar dados relativos a preços de combustíveis no Brasil de forma que os usuários possam selecionar onde podem abastecer, levando em consideração seus trajetos e o custo benefício das rotas possíveis em seu caminho. Além de mostrar dados estatisticos de forma mais intuitivas e de forma didática.
